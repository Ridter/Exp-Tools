## 简介
该工具使用了ExpDemo-JavaFX项目，保留了核心的数据包请求接口。对系列oa漏洞进行了复现和分析，极力避免exp的误报和有效性。

截止到目前为止，已实现了用友、泛微、通达、致远、帆软报表、万户、蓝凌、红帆、华天动力总共9个OA。
全部是命令执行、文件上传类的漏洞，包括前台和后台，已知漏洞和未知漏洞，未编写log4j、fastjson相关漏洞。
